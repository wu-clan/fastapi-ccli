# fastapi 项目克隆命令行工具

> Ps: 浅用 [Typer](https://typer.tiangolo.com/)

## install
```shell
git clone https://gitee.com/wu_cl/fastapi_project_clone_cli
```

## use
```shell
pip install -r requirements.txt

python main.py
```

## help
```shell
python main.py --help
```

## Build
如果你想将它打包, 可以查看此[文档](https://typer.tiangolo.com/tutorial/package/)
